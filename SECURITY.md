# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this website or any of the games
hosted here, please report it privately rather than opening a public issue.

- **Preferred:** Use GitHub's private vulnerability reporting on this repository
  (Security tab → "Report a vulnerability").
- Please include steps to reproduce and, if possible, the affected page or game.

We aim to acknowledge reports within a week. Thank you for helping keep
Wild Olive Studios players safe!

## Scope

This repository contains a static GitHub Pages site and browser games. There is
no backend, no user accounts, and no data is collected or transmitted — game
saves live entirely in your browser's localStorage.
